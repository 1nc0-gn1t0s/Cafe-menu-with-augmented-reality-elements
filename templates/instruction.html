Создание аккаунта

Работа с личным кабинетом не может совершаться с телефона. "С нуля" создается только аккаунт администратора. Аккаунт менеджера создается во время создания нового филиала определенной ресторанной сети через аккаунт администратора. Для создания аккаунта нажмите на "Регистрация".

Введите информацию о себе. Пароль должен состоять как минимум из 8 элементов, он не может содержать пробелов, в нем должна быть хотя бы одна заглавная и хотя бы одна строчная буквы на латинице. Пароль также должен содержать как минимум одну цифру.

Вам нужно будет подтвердить почту и ввести код, присланный Вам на почту. Если во время подтверждения почты позникает ошибка, то убедитесь, что VPN на Вашем устройстве выключен.

Вход в аккаунт

Выберите должность. На одну и ту же почту может быть сделано два аккаунта: аккаунт администратора и аккаунт менеджера.

Введите данные и войдите в аккаунт. Если Вы не помните пароль, то Вы сможете его изменить. Вам отправят код на почту для подтверждения.

Аккаунт администратора

В аккаунте администратора можно создавать ресторанные сети, добавлять блюда и филиалы.

Создание ресторанной сети

Заполните все необходимые данные. Окна со звездочками являются обязательными.

Создание филиала ресторанной сети

Из аккаунта администратора переходим в "Филиалы" нужного ресторана.

Вы можеле удалить или добавить филиал. Для удаления филиала нажмите "Удалить филиал". Чтобы добавить филиал, нажмите на "Добавить филиал" и заполните нужную информацию.

Зарегистрируйте менеджера. Ему на почту прийдет письмо с паролем и ссылкой на авторизацию.

Создание блюда

Для просмотра меню ресторанной сети нажмите на "Меню" со страницы аккаунта администратора.

Для создания нового блюда нажмите "Добавить блюдо", для удаления -- "Удалить блюдо".

Заполните всю необходимую информацию. Поддерживаемые форматы изображений: .jpeg, .jpg и .png. Файлы AR могут быть фармата .gltf или .glb. Размеры блюда необходимы для корректного отображения AR-меню. Аккаунт менеджера

На аккаунт менеджера приходят уведомления от пользователей.

Менеджер может удалить уведомления по нажатию кнопки "Удалить".

Настройка аккаунта

Нажмите на "Настройка аккаунта" со страницы пользователя. Заполните поля, которые хотите изменить. Вам не обязательно заполнять все.

Стандартное меню

В обычном меню можно посмотреть рестораны и их блюда, которые будут отображаться с фотографиями и без возможности посмотреть AR.

Меню в режиме дополненной реальности

При переходе по ссылке AR-меню для филиала (ссылку можно найти в описании филиала через аккаунт администратора) открывается страница ввода номера столика.

Пользователю необходимо ввести номер своего столика корректно для удобства работы официантов.

далее пользователь может прочитать информацию о каждом блюде и посмотреть эти блюда в режиме дополненной реальности по нажатию соответствующей кнопки. Пользователь может позвать официанта по кнопке в верхнем правом углу экрана.
